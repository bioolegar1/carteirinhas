# Fotos das carteirinhas

Para a primeira carteirinha, coloque a foto neste caminho:

```text
public/assets/photos/propria-carteirinha.jpg
```

A aplicacao ja aponta o primeiro cadastro para:

```text
/assets/photos/propria-carteirinha.jpg
```

Enquanto a imagem nao existir, a carteirinha mostra o placeholder.
